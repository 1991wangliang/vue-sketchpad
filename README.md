# sketchpad

>使用vue.js和fabric.js构建的web画板，目前支持画箭头、虚线、矩形、圆型、三角形、文字等功能

效果图如下：
!["效果图"](src/assets/img.png)

[在线预览](blog.hjplay.cn/vue-sketchpad/)

- [x] 直线
- [x] 箭头
- [x] 虚线
- [x] 输入文字
- [x] 矩形
- [x] 标准圆形
- [x] 椭圆形
- [x] 等腰三角形
- [x] 删除、移动、复位
- [x]  历史记录
